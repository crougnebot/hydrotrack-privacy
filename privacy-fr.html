Politique de confidentialité

HydroTrack — Suivi quotidien de consommation d'eau

Dernière mise à jour : 10 avril 2026

HydroTrack (« nous », « notre », « l'Application ») s'engage à protéger votre vie privée. Cette politique explique quelles données nous collectons, comment nous les utilisons et vos droits.

1. Informations que nous collectons

1.1 Données que vous fournissez

  • Données de consommation d'eau : Vos entrées quotidiennes de consommation d'eau. Ces données sont stockées localement sur votre appareil via Hive (une base de données locale). Si vous activez la synchronisation cloud, ces données peuvent également être stockées dans Firebase Firestore sous un identifiant utilisateur anonyme.
  • Préférences : Objectifs d'hydratation quotidiens, paramètres de notification, préférences d'affichage (mode sombre, unités).

1.2 Données collectées automatiquement

  • Firebase Analytics : Nous collectons des données d'utilisation anonymes telles que les vues d'écran, la fréquence d'utilisation des fonctionnalités et la durée des sessions. Aucune information personnellement identifiable n'est collectée.
  • Firebase Crashlytics : En cas de crash de l'application, nous collectons les journaux de crash, le modèle d'appareil, la version du système et les traces d'exécution pour diagnostiquer et résoudre les problèmes. Ces données sont anonymisées.
  • AdMob (Google Mobile Ads) : Si des publicités sont affichées, Google peut collecter des identifiants d'appareil (par ex. l'identifiant publicitaire) pour la personnalisation des annonces. Vous pouvez désactiver les annonces personnalisées dans les paramètres de votre appareil.

1.3 Données que nous ne collectons PAS

  • Nous ne collectons pas votre nom, adresse e-mail, numéro de téléphone ou tout autre identifiant personnel.
  • Nous ne collectons pas de données de localisation.
  • Nous n'accédons pas à vos contacts, photos ou autres applications.
  • Nous ne vendons aucune donnée à des tiers.

2. Comment nous utilisons vos données

  • Données de consommation d'eau : Pour afficher votre historique d'hydratation, vos graphiques et vos progrès vers vos objectifs.
  • Données analytiques : Pour comprendre comment les utilisateurs interagissent avec l'application et améliorer les fonctionnalités.
  • Données de crash : Pour identifier et corriger les bugs.
  • Données publicitaires : Pour afficher des publicités pertinentes (gérées par Google AdMob).

3. Stockage et sécurité des données

  • Stockage local : Vos données de consommation d'eau sont stockées sur votre appareil via Hive, une base de données locale rapide.
  • Synchronisation cloud (optionnelle) : Si activée, les données sont synchronisées vers Firebase Firestore via HTTPS (chiffrées en transit). Les règles de sécurité Firebase protègent l'accès.
  • Mesures de sécurité : Toute communication réseau utilise HTTPS. L'infrastructure Firebase offre une sécurité de niveau professionnel.

4. Partage des données

Nous ne partageons pas vos données avec des tiers, sauf :

  • Google Firebase : Pour les analyses, les rapports de crash et la synchronisation cloud optionnelle (données anonymisées).
  • Google AdMob : Pour la diffusion de publicités (utilise l'identifiant publicitaire selon les politiques Google Play).

5. Vos droits

  • Accéder à vos données : Vous pouvez consulter toutes vos données dans l'application à tout moment.
  • Supprimer vos données : Allez dans Paramètres → Effacer toutes les données pour supprimer définitivement votre historique de consommation et vos préférences. Si la synchronisation cloud est activée, la suppression locale supprime également les données cloud.
  • Désactiver les publicités personnalisées : Désactivez les annonces personnalisées dans les paramètres Google de votre appareil → Annonces → Désactiver la personnalisation des annonces.
  • Exporter vos données : Vous pouvez exporter vos données depuis les paramètres de l'application.

6. Confidentialité des enfants

HydroTrack ne collecte pas sciemment de données auprès d'enfants de moins de 13 ans. L'application convient à tous les âges et ne nécessite pas de création de compte.

7. Modifications de cette politique

Nous pouvons mettre à jour cette politique de temps à autre. Les modifications seront indiquées par la date de « Dernière mise à jour » ci-dessus.

8. Contact

Si vous avez des questions concernant cette politique de confidentialité, contactez-nous à :
E-mail : privacy@crougne.com
